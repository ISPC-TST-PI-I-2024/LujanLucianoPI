Ejercicio 2: Lectura de Sensores Resistivos

Objetivos:

Aprender a leer valores analógicos de un potenciómetro.

Convertir valores analógicos en datos digitales comprensibles.

Visualizar y monitorear cambios en el valor del potenciómetro a través del monitor serie.

Simular la conexión y lectura de un sensor resistivo en Proteus.

Desarrollo:

1. Conectar un potenciómetro a un pin analógico del Arduino:

Conecte un extremo del potenciómetro al pin de entrada analógica del Arduino (por ejemplo, pin A0).

Conecte el otro extremo del potenciómetro a la alimentación de 5V del Arduino.

Conecte el terminal central del potenciómetro a tierra (GND) del Arduino.

2. Utilizar la función analogRead() para leer los valores del potenciómetro:

C++
void setup() {
  Serial.begin(9600);
}

void loop() {
  // Leer el valor del potenciómetro
  int sensorValue = analogRead(A0);

  // Convertir el valor analógico a un rango de 0 a 100
  float voltage = sensorValue * (5.0 / 1023.0);
  float percentage = (voltage / 5.0) * 100.0;

  // Mostrar los valores en el monitor serie
  Serial.print("Valor analógico: ");
  Serial.println(sensorValue);
  Serial.print("Voltaje: ");
  Serial.println(voltage, 2);
  Serial.print("Porcentaje: ");
  Serial.println(percentage, 2);
  Serial.println();

  delay(100); // Esperar 100 milisegundos antes de la siguiente lectura
}
Usa el código con precaución.
